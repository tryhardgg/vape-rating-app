# 🗄️ Схема базы данных Supabase

**Проект:** Monster Vapor Tracker  
**Таблица:** `cards`  
**Последнее обновление:** 2026-04-01

---

## 📊 Структура таблицы

```sql
CREATE TABLE cards (
    id              BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    card_id         TEXT UNIQUE NOT NULL,
    flavor          TEXT,
    note            TEXT,
    image           TEXT,
    rating_sasha    DOUBLE PRECISION,
    rating_nastya   DOUBLE PRECISION,
    emojis          JSONB DEFAULT '[]'::jsonb,
    is_filled       BOOLEAN DEFAULT FALSE,
    is_other        BOOLEAN DEFAULT FALSE,
    created_at      TIMESTAMPTZ DEFAULT NOW(),
    updated_at      TIMESTAMPTZ DEFAULT NOW()
);
```

---

## 📋 Описание полей

| Поле | Тип | Nullable | Default | Описание |
|------|-----|----------|---------|----------|
| `id` | BIGINT | ❌ | AUTO | Первичный ключ (автоинкремент) |
| `card_id` | TEXT | ❌ | - | Уникальный ID карточки (timestamp) |
| `flavor` | TEXT | ✅ | NULL | Название вкуса (например, "Barawolf") |
| `note` | TEXT | ✅ | NULL | Примечание пользователя |
| `image` | TEXT | ✅ | NULL | Изображение (base64 или URL) |
| `rating_sasha` | FLOAT8 | ✅ | NULL | Рейтинг Саши (0-10) |
| `rating_nastya` | FLOAT8 | ✅ | NULL | Рейтинг Насти (0-10) |
| `emojis` | JSONB | ✅ | `[]` | Массив выбранных иконок |
| `is_filled` | BOOLEAN | ✅ | FALSE | Заполнена ли карточка |
| `is_other` | BOOLEAN | ✅ | FALSE | Вкладка "Эксперименты" |
| `created_at` | TIMESTAMPTZ | ✅ | NOW() | Дата создания |
| `updated_at` | TIMESTAMPTZ | ✅ | NOW() | Дата обновления |

---

## 🔑 Индексы

```sql
-- Первичный ключ (автоматически)
ALTER TABLE cards ADD PRIMARY KEY (id);

-- Уникальный индекс на card_id
CREATE UNIQUE INDEX IF NOT EXISTS idx_cards_card_id ON cards(card_id);

-- Индекс для быстрого поиска по is_other
CREATE INDEX IF NOT EXISTS idx_cards_is_other ON cards(is_other);

-- Индекс для сортировки по updated_at
CREATE INDEX IF NOT EXISTS idx_cards_updated_at ON cards(updated_at DESC);
```

---

## 🔐 RLS Policy (Row Level Security)

```sql
-- Включить RLS
ALTER TABLE cards ENABLE ROW LEVEL SECURITY;

-- Разрешить все операции всем
CREATE POLICY "Доступ всем" ON cards
FOR ALL USING (true) WITH CHECK (true);
```

**Почему так:**
- Приложение публичное (только 2 пользователя)
- Все данные общие
- Нет чувствительной информации

## 📝 Примеры запросов

### SELECT (чтение)

```sql
-- Все карточки
SELECT * FROM cards;

-- Только Monster Vapor
SELECT * FROM cards WHERE is_other = FALSE;

-- Только Эксперименты
SELECT * FROM cards WHERE is_other = TRUE;

-- Заполненные карточки
SELECT * FROM cards WHERE is_filled = TRUE;

-- С рейтингами
SELECT 
    flavor,
    rating_sasha,
    rating_nastya,
    (rating_sasha + rating_nastya) / 2 AS average_rating
FROM cards
WHERE is_filled = TRUE
ORDER BY average_rating DESC;
```

### INSERT (вставка)

```sql
-- Новая карточка
INSERT INTO cards (
    card_id,
    flavor,
    rating_sasha,
    rating_nastya,
    emojis,
    is_filled,
    is_other
) VALUES (
    '1775000000000',
    'Test Flavor',
    8.5,
    9.0,
    '["⚠️", "🌪"]'::jsonb,
    TRUE,
    FALSE
);
```

### UPDATE (обновление)

```sql
-- Обновить рейтинг
UPDATE cards
SET 
    rating_sasha = 9.0,
    rating_nastya = 8.5,
    updated_at = NOW()
WHERE card_id = '1775000000000';

-- Обновить иконки
UPDATE cards
SET 
    emojis = '["⚠️", "🌪", "💎"]'::jsonb,
    updated_at = NOW()
WHERE card_id = '1775000000000';
```

### DELETE (удаление)

```sql
-- Удалить карточку
DELETE FROM cards
WHERE card_id = '1775000000000';

-- Удалить все Эксперименты
DELETE FROM cards
WHERE is_other = TRUE;
```

## 🔍 Представления (Views)

### Средние рейтинги

```sql
CREATE VIEW v_card_ratings AS
SELECT 
    card_id,
    flavor,
    rating_sasha,
    rating_nastya,
    ROUND(((rating_sasha + rating_nastya) / 2)::NUMERIC, 1) AS average_rating,
    is_other
FROM cards
WHERE is_filled = TRUE;
```

### Использование:

```sql
-- Топ 10 по рейтингу
SELECT * FROM v_card_ratings
ORDER BY average_rating DESC
LIMIT 10;
```

## 📊 Статистика

```sql
-- Количество карточек
SELECT COUNT(*) FROM cards;

-- Заполненные карточки
SELECT COUNT(*) FROM cards WHERE is_filled = TRUE;

-- Эксперименты
SELECT COUNT(*) FROM cards WHERE is_other = TRUE;

-- Средний рейтинг по всем
SELECT 
    ROUND(AVG(rating_sasha), 2) AS avg_sasha,
    ROUND(AVG(rating_nastya), 2) AS avg_nastya,
    ROUND(AVG((rating_sasha + rating_nastya) / 2), 2) AS avg_total
FROM cards
WHERE is_filled = TRUE;
```

---

## ⚠️ Важные замечания

1. **`card_id` хранится как TEXT** (не BIGINT) для совместимости с JavaScript
2. **`emojis` — JSONB массив** строк (например, `["⚠️", "🌪"]`)
3. **`rating_*` — FLOAT8** (допускает 0.5 шаг: 7.5, 8.0, 9.5)
4. **`updated_at`** обновляется автоматически при каждом изменении
